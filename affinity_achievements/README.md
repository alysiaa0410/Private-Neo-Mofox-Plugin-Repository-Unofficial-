## 亲密度与成就系统插件（affinity_achievements）

一个为 MoFox-Bot 设计的轻量“亲密度 + 成就”系统插件示例。

### 功能特性

- **亲密度累计**：基于用户在本进程内与 Bot 的交互次数，粗略统计亲密度。
- **成就称号**：到达不同消息数阈值时，解锁不同称号（如“常来坐坐”“老朋友”等）。
- **即时反馈**：通过指令查看当前亲密度等级和下一目标，增强陪伴感和游戏性。

> 注意：本示例插件仅在内存中保存数据，重启后记录会清零，如需持久化可自行接入数据库。

### 指令说明

- `/affinity`  
  查看当前亲密度信息，包括：
  - 累计对话条数
  - 当前称号
  - 已解锁成就列表
  - 下一目标说明（还差多少条对话解锁下一级）

### 权限节点

- `affinity_achievements.use`：允许用户使用 `/affinity` 指令查看亲密度与成就。
