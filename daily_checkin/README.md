## 每日签到与连击插件（daily_checkin）

### 功能

为 MoFox-Bot 增加一个轻量的每日签到系统，记录用户当日是否签到，并给出连击天数和简单奖励提示。

### 指令

- `/checkin` - 今日签到
- `/checkin status` - 查看当前连击天数

### 权限节点

- `daily_checkin.use` - 允许使用签到功能

### 使用示例

```
/checkin
✅ 签到成功！今天是你连续签到的第 1 天。
从今天开始，我们一起养成一个小小的好习惯。

/checkin status
你当前的连续签到天数为：7 天。
上次签到日期：2026-02-27
```
