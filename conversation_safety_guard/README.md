## 对话情绪分析与安全建议插件（conversation_safety_guard）

### 功能

为 MoFox-Bot 提供基础的情绪倾向分析与安全建议功能，通过指令分析一段话的情绪（如积极、中性、消极、可能高风险），并给出温和的应对建议。

### 指令

- `/mood <文本内容>` - 分析文本的情绪倾向

### 权限节点

- `conversation_safety_guard.use` - 允许使用情绪分析功能

### 使用示例

```
/mood 我最近压力有点大，总是睡不着
🧠 情绪检测结果：
- 粗略判断：偏消极 / 低落情绪
- 情绪强度：80 / 100

💡 建议：可以适当给自己一点空间和时间...
```
