## 多角色预设与场景切换插件（scene_persona_switcher）

一个为 MoFox-Bot 设计的“人设/场景预设与切换”插件示例。

### 功能特性

- **多种预设场景**：内置“学习助手模式”“吐槽搭子模式”“严肃顾问模式”等。
- **一键切换人设**：通过指令切换当前会话的人设场景。
- **Prompt 建议文本**：每个场景会返回一段推荐的提示词，可直接用于 KFC/AFC 配置或其他 Prompt 注入插件。

### 指令说明

- `/scene list`  
  查看当前可用场景列表及简介。

- `/scene 学习`  
  切换到“学习助手模式”，适合讲解知识、刷题、学习相关任务。

- `/scene 吐槽`  
  切换到“吐槽搭子模式”，更口语和放松，适合日常聊天和吐槽。

- `/scene 严肃`  
  切换到“严肃顾问模式”，表达相对严谨，适合规划、分析和决策场景。

### 使用方式

切换成功后，插件会返回：

- 场景名称与简介
- 一段推荐的人设 Prompt 文本（多行），你可以：
  - 复制到 MoFox 的 KFC/AFC 配置中；
  - 或配合 `custom_prompt_injector` 等插件进行自动注入。

### 权限节点

- `scene_persona_switcher.use`：允许用户使用 `/scene` 指令查看与切换场景。
