## 多角色预设与场景切换插件（scene_persona_switcher）

### 功能

为 MoFox-Bot 提供多套可配置的人设与场景预设（如学习助手、日常聊天、严肃顾问等），通过指令一键切换当前会话使用的提示词与语气说明。

### 指令

- `/scene list` - 查看可用场景
- `/scene <场景名>` - 切换到指定场景

### 权限节点

- `scene_persona_switcher.use` - 允许使用场景切换功能

### 预设场景

1. **学习** - 学习助手模式，适合讲解知识、刷题、辅导学习任务
2. **吐槽** - 吐槽搭子模式，适合轻松聊天、吐槽日常
3. **严肃** - 严肃顾问模式，适合做决策分析、项目规划

### 使用示例

```
/scene list
📑 当前可用场景：
- 学习：学习助手模式 —— 适合讲解知识、刷题、辅导学习任务。
- 吐槽：吐槽搭子模式 —— 适合轻松聊天、吐槽日常，风格更口语化。
- 严肃：严肃顾问模式 —— 适合做决策分析、项目规划等，需要更稳重的表达风格。

/scene 学习
✅ 已切换到场景：学习助手模式
简介：适合讲解知识、刷题、辅导学习任务。
```
