## 对话简要回顾插件（conversation_recap）

一个为 MoFox-Bot 设计的“近期对话回顾”插件示例。

### 功能特性

- **快速回顾**：从当前会话中读取最近若干条消息，整理成一份简要清单。
- **结构化输出**：按时间顺序列出“你 / Bot”说过的内容，方便中途离开再回来时快速回忆上下文。
- **实现透明**：不依赖大模型做自动摘要，逻辑简单易读，便于二次开发。

### 指令说明

- `/recap`  
  插件会：
  - 从当前会话中读取最近 N 条消息（默认 30 条）；
  - 过滤出包含文本内容的消息；
  - 生成类似：

```text
📚 近期对话小结（仅基于最近若干条消息）：
- 你：今天帮我看看配置文件
- Bot：好的，把你的报错信息发一下
- 你：报错是 xxx...
...
```

### 权限节点

- `conversation_recap.use`：允许用户使用 `/recap` 指令生成对话回顾。
