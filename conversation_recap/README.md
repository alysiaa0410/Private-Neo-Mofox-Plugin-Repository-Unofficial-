## 对话简要回顾插件（conversation_recap）

### 功能

为 MoFox-Bot 提供一个简单的对话回顾功能，可以根据近期消息生成一份结构化的小结，帮助用户快速回忆本次对话的大致内容。

### 指令

- `/recap` - 生成最近若干条对话的小结

### 权限节点

- `conversation_recap.use` - 允许使用对话回顾功能

### 使用示例

```
/recap
📝 正在整理最近的对话内容，请稍候...
📚 近期对话回顾（基于最近若干条消息）：
- 你： 你好
- Bot： 你好！有什么我可以帮助你的吗？
- 你： 今天天气怎么样
- Bot： 抱歉，我无法获取实时天气信息...
```
