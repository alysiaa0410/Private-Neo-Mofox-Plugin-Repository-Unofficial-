## 每日签到与连击插件（daily_checkin）

一个为 MoFox-Bot 设计的“每日签到 + 连击天数”插件示例。

### 功能特性

- **每日签到**：用户每天可以通过指令完成一次签到。
- **连击统计**：记录连续签到天数，根据中断与否自动重置或递增。
- **鼓励文案**：不同连击区间返回不同的鼓励语，增强打卡动力。

> 说明：本示例插件仅在内存中保存签到状态，重启后会清空，如需长期使用可接入数据库。

### 指令说明

- `/checkin`  
  今日签到。如果今天尚未签到：
  - 记录签到日期并更新连击天数；
  - 返回当前连击天数和一段对应的鼓励文案。

- `/checkin status`  
  查看当前连击天数和最近一次签到日期。

### 权限节点

- `daily_checkin.use`：允许用户使用 `/checkin` 指令进行签到与查看状态。
