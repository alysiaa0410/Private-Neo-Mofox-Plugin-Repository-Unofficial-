## 对话情绪分析与安全建议插件（conversation_safety_guard）

一个为 MoFox-Bot 设计的轻量级情绪分析与安全建议插件示例。

### 功能特性

- **基础情绪判断**：通过简单关键词，粗略判断文本是偏积极、中性、消极还是可能较高风险。
- **安全建议**：根据不同情绪类型给出对应的安抚和行动建议。
- **指令触发**：不会自动拦截所有对话，而是通过命令显式调用，适合作为“安全工具箱”组件。

### 指令说明

- `/mood 文本内容`  
  或：`心情检测 文本内容`

示例：

```text
/mood 我最近总是睡不着，感觉有点撑不住了
```

插件会返回：

- 粗略的情绪类型（偏积极 / 偏消极 / 可能高风险等）
- 情绪强度分值（0–100）
- 一段温和的建议文字和安全说明

### 权限节点

- `conversation_safety_guard.use`：允许用户使用 `/mood` 情绪分析指令。
