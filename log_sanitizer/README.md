## 日志隐私脱敏插件（log_sanitizer）

一个为 MoFox-Bot 设计的日志隐私脱敏插件示例。

### 功能特性

- **基础脱敏规则**：对文本中的手机号、邮箱、可能的身份证号片段进行掩码处理。
- **便于调试**：可以直接在聊天中试验脱敏效果，为后续在日志系统中接入类似逻辑做准备。
- **纯标准库实现**：只依赖 Python 正则库，方便理解与扩展。

### 指令说明

- `/sanitize 文本内容`  
  例如：

```text
/sanitize 我的手机号是 13812345678，邮箱 test@example.com
```

插件会返回：

- 原文
- 脱敏后的文本（如手机号中间四位变为 *，邮箱用户名部分做模糊）

### 扩展建议

- 你可以将 `sanitize_text` 函数抽取出来，在自己的日志中间件中调用，实现真正的日志脱敏。
- 也可以根据实际需要扩充更多正则规则，如银行卡号、地址等。

### 权限节点

- `log_sanitizer.use`：允许用户使用 `/sanitize` 指令进行文本脱敏。
