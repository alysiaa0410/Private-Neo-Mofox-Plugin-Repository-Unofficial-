## 轻量 FAQ 与知识库增强插件（simple_faq_knowledge）

一个为 MoFox-Bot 设计的超轻量 FAQ / 知识库插件示例。

### 功能特性

- **内置 FAQ 列表**：使用纯 Python 字典维护若干常见问题及标准答案。
- **关键词匹配**：通过简单的“关键词命中 + 字符集相似度”找到最合适的 FAQ 条目。
- **标准回复**：对命中的问题返回稳定一致的标准答案，降低大模型调用成本。

### 指令说明

- `/faq 关键字...`  
  例如：

```text
/faq 帮助
/faq 插件 列表
```

插件会根据关键词和相似度，返回一条最接近的问题及其标准答案。

### 配置与二次开发

- 你可以打开 `plugin.py`，在 `BUILT_IN_FAQ` 列表中添加、修改或删除 FAQ 条目。
- 每个条目包含：
  - `question`: 问题标题
  - `keywords`: 若干关键词，命中时会提高匹配得分
  - `answer`: 标准答案文本

### 权限节点

- `simple_faq_knowledge.use`：允许用户使用 `/faq` 指令查询 FAQ。
