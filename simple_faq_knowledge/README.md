## 轻量 FAQ 与知识库增强插件（simple_faq_knowledge）

### 功能

为 MoFox-Bot 增加一个基于内置配置的轻量 FAQ/知识库系统，对常见问题给出稳定一致的标准答案，降低大模型调用成本并提升可控性。

### 指令

- `/faq <关键字>` - 查询常见问题

### 权限节点

- `simple_faq_knowledge.use` - 允许使用FAQ查询功能

### 使用示例

```
/faq 帮助
📚 FAQ 匹配结果：

❓ 如何获取帮助或查看常用指令？

💡 标准答案：
可以尝试发送 `/help` 或在项目文档中查找"常用指令"章节...
```

### 内置FAQ条目

1. 如何查看当前可用插件？
2. 如何获取帮助或查看常用指令？
3. 遇到问题应该如何反馈？
